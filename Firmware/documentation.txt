Circuit Documentation
Summary
This circuit is designed to monitor environmental conditions using an ESP32-S3 microcontroller. It integrates various sensors and output devices to measure distance, detect rain, and provide visual and auditory alerts. The circuit also includes an OLED display for real-time data visualization and a WiFi module for sending data to a remote server.

Documented Code
Overview
The code is written for the ESP32-S3 microcontroller and is responsible for reading sensor data, controlling LEDs and a piezo speaker, displaying information on an OLED screen, and sending data to a remote server via WiFi.
Key Features
WiFi Connectivity: Connects to a WiFi network and sends data to a specified server URL.
Sensor Reading: Reads distance from the ultrasonic sensor and rain status from the rain sensor.
Alert System: Uses LEDs and a piezo speaker to indicate different alert levels based on sensor readings.
OLED Display: Displays real-time data including rain status, distance, water level, and alert status.
Multitasking: Utilizes FreeRTOS to run tasks concurrently on different cores of the ESP32-S3.
